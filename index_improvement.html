The preliminary pages for the design

"The home page of the project resembles some basic instructions about how to use the web application. It is build in order to be intuitive, so the user would only need to see this first page to understand what to do"

"The default map page is accesible directly through leaflet API, so it can display different basemaps for a better user interface experience. The user might find more productive to use a different basemap everytime is analyzing the viewshed of an area"

"The main interface to use the web application helps the user to define a base point (the place from which the views need to be accesible), then the user is able to define the places to view. Finally,the results can be upload or download as the user desires."
